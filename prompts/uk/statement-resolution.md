# Правила для тверджень

При аналізі позиції тексту щодо твердження (висловлювання, key message):

Твердження — це конкретне судження або теза, яка може бути підтверджена, спростована або уточнена текстом. Приклади:
- "Зміна клімату — реальна загроза"
- "Використовувати російські соціальні мережі небезпечно"
- "Цифровізація поліпшує якість життя"

Враховуйте такі правила:

1. **Пряме підтвердження/спростування**: Текст прямо згоден або не згоден з твердженням
   - Приклад твердження: "Клімат змінюється через діяльність людини"
   - Згода: "Антропогенні викиди прискорюють глобальне потепління"
   - Незгода: "Кліматичні цикли природні та не залежать від людини"

2. **Уточнення/модифікація**: Текст додає умови, обмеження або нюанси
   - Приклад: "Клімат змінюється, але це не головна проблема"
   - Приклад: "Соціальні мережі небезпечні, але тільки при неправильному використанні"

3. **Контекстуальна інтерпретація**: Текст не згадує твердження явно, але його зміст передбачає позицію
   - Твердження: "Цифровізація поліпшує якість життя"
   - Текст: "Онлайн-освіта доступніша, ніж очна" → Непряма згода
   - Текст: "Люди втрачають навички спілкування через інтернет" → Непряма незгода

4. **Релевантність**: Визначте, чи стосується текст твердження
   - Якщо текст явно не пов'язаний з твердженням → Нейтральна позиція (позатематичний)
   - Якщо текст частково пов'язаний, але не виражає позицію → Нейтральна позиція
   - Якщо текст явно суперечить або підтверджує → Позитивна/Негативна
   
   Приклади:
   - Твердження: "Використовувати російські соцмережі небезпечно"
   - Нерелевантний текст: "Я люблю готувати пасту" → Нейтральна (позатематичний)
   - Частково пов'язаний: "ВКонтакте зручніше, ніж Facebook" → Нейтральна (не виражає позицію щодо небезпеки)
   - Релевантний: "ВКонтакте передає ваші чати товаришу майору" → Позитивна (підтримує твердження про небезпеку)

5. **Інтенсивність та емоційність**: Звертайте увагу на силу вираження позиції
   - Слаба згода: "Можливо, клімат змінюється"
   - Сильна згода: "Клімат катастрофічно змінюється!"
   - Амбівалентність: "Клімат змінюється, але це не так погано" → Може бути нейтральною

6. **Мовчазна згода/незгода**: Поведінка або дії, які передбачають позицію
   - Твердження: "Російські соцмережі небезпечні"
   - Текст: "Я активно використовую ВКонтакте" → Може бути нейтральною (поведінка ≠ позиція)
   - Текст: "Я активно використовую ВКонтакте, тому що це безпечно" → Негативна (явна позиція)
   - Текст: "Я уникаю ВКонтакте через проблеми з приватністю" → Позитивна (підтримує твердження)

7. **Модальності**: Слова та фрази, які кваліфікують ставлення до твердження:
   
   - **Згода**: "дійсно", "очевидно", "звичайно", "вірно", "правильно", "згоден"
   - **Незгода**: "але", "однак", "навпаки", "не згоден", "неправда", "помилка"
   - **Невпевненість**: "може бути", "можливо", "здається", "схоже", "ймовірно"
   
   - **Посилання на думку (без особистої позиції)**: "депутат заявив, що", "експерти вважають, що", "на думку", "кажуть, що"
     - Класифікація: Нейтральна (при відсутності виразної позиції автора тексту)
     - Виключення: Якщо автор додає оцінку, це стає позицією
   
   - **Оціночна модальність**: Слова, які несуть приховану оцінку
     - Позитивна оцінка: "провідні фахівці", "авторитети", "визнані експерти", "правильно підтримують"
     - Негативна оцінка: "зрадник", "брехун", "шарлатан", "неправильно стверджують"
     - Класифікація: Може бути Позитивною/Негативною залежно від контексту
   
   Приклади:
   - "Клімат змінюється, але це природний процес" → Незгода (сигнал "але")
   - "Дійсно, соцмережі небезпечні" → Згода (сигнал "дійсно")
   - "Можливо, цифровізація поліпшує життя" → Невпевненість (сигнал "можливо")
   - "Експерти вважають, що клімат змінюється" → Нейтральна (посилання на думку без особистої позиції)
   - "Провідні кліматологи підтверджують: клімат змінюється" → Позитивна (оціночна модальність + згода)
   - "Зрадник Медведчук заявив, що соцмережі безпечні" → Негативна (оціночна модальність "зрадник" + незгода)

8. **Заперечення**: Будьте уважні до заперечних конструкцій
   - Твердження: "Соціальні мережі корисні"
   - Текст: "Соціальні мережі НЕ корисні" → Негативна (явне заперечення)
   - Текст: "Соціальні мережі не завжди корисні" → Нейтральна (кваліфікація, не повне заперечення)
   - Текст: "Соціальні мережі не такі корисні, як здається" → Негативна (послаблена, але все ж негативна)
   
9. **Іронія та сарказм**: Будьте уважні до іронії та сарказму
   - Твердження: "Соціальні мережі корисні"
   - Текст: "Так, соцмережі дуже корисні для нашого психічного здоров'я" (сарказм) → Негативна
   - Текст: "Звичайно, ми всі знаємо, що соцмережі безпечні" (іронія) → Негативна
   
   Сигнали іронії (варто розрізняти від простої експресивності):
   - Перебільшення: "Соцмережі просто РЯТУЮТЬ наше життя!"
   - Контраст: "Так, соцмережі дуже корисні" + негативний контекст
   - Лапки: "Соцмережі 'безпечні'" (лапки вказують на сумнів)

10. **Умовні висловлювання**: Текст виражає позицію умовно
    - Твердження: "Цифровізація поліпшує якість життя"
    - Текст: "Якщо правильно використовувати цифровізацію, вона поліпшує життя" → Позитивна (умовна)
    - Текст: "Цифровізація поліпшує життя, якщо не враховувати побічні ефекти" → Нейтральна (умовна з застереженнями)
    
    Класифікація:
    - Якщо умова реалістична → Позитивна/Негативна
    - Якщо умова малоймовірна → Нейтральна (приховане несогласие)

|---|---|---|---|

| "Клімат змінюється через людину" | "Кліматичні цикли — природний процес" | Негативна | Пряме спростування |
| "Соцмережі небезпечні" | "Соцмережі корисні для бізнесу, але шкодять психіці" | Нейтральна | Амбівалентна позиція |
| "Цифровізація поліпшує життя" | "Молодь не може жити без інтернету" | Нейтральна | Констатація факту, не оцінка |
| "Російські соцмережі небезпечні" | "Я активно використовую ВКонтакте" | Нейтральна | Поведінка не передбачає позицію |
