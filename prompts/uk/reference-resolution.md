# Рекомендації щодо розв'язання референцій

## Основний принцип

При аналізі позиції автора тексту щодо цільового об'єкта використовуй 
всі доступні контекстуальні сигнали: займенники, еліпсис, синоніми 
та підразумівані посилання. Явне називання об'єкта НЕ вимагається.

## Коли розв'язувати референцію

**Розв'язуй референцію, якщо:**

1. Об'єкт явно згаданий у тексті (пряме посилання)
2. Об'єкт підразумівається через займенник або еліпсис 
   (приклад: "Це небезпечно" → об'єкт зрозумілий з контексту)
3. Об'єкт логічно виводиться з теми обговорення 
   (приклад: аналізуємо ставлення до "мови меншин", текст говорить про 
   "загрозу національній ідентичності" → це ясне посилання на мову меншин)
4. Можна допустити, що об'єкт присутній у контексті публікації 
   (наприклад, у заголовку, зображенні, попередніх коментарях у ланцюжку) 
   → **презумпція присутності**: якщо немає явних підстав заперечувати 
   присутність об'єкта, вважай його присутнім

## Коли відмовляти у розв'язанні

**Відмовляй розв'язувати референцію, якщо:**

1. Припущення про референцію вимагає значного домислу або фонових знань, 
   недоступних із тексту
2. Об'єкт суперечить явному змісту тексту

## Приклади

### Приклад 1: Розв'язання через займенник

```
Цільовий об'єкт: "Санкції проти Росії"
Текст: "Санкції проти Росії вводять щодня. [ще 2-3 речення] 
Це руйнує економіку України"
Розв'язання: ✅ "Це" = санкції проти Росії (з контексту)
```

### Приклад 2: Розв'язання через еліпсис

```
Цільовий об'єкт: "Патріотичне виховання"
Текст: "Депутат Бровко наче підтримує це, але його діти живуть у Парижі"
Розв'язання: ✅ "це" = патріотичне виховання (логічно)
```

### Приклад 3: Розв'язання через синонім

```
Цільовий об'єкт: "VPN"
Текст: "Ці інструменти для обходу блокувань небезпечні"
Розв'язання: ✅ "інструменти" = VPN (логічний висновок)
```

### Приклад 4: Презумпція присутності

```
Цільовий об'єкт: "Євроінтеграція"
Текст: "Це неминуче і правильно"
Розв'язання: ✅ "Це" = євроінтеграція (немає підстав заперечувати присутність об'єкта в контексті)
```

### Приклад 5: Відмова у розв'язанні

```
Цільовий об'єкт: "Цифрова трансформація"
Текст: "Погода сьогодні гарна"
Розв'язання: ❌ Немає зв'язку між текстом та об'єктом
```
